Software

Below are a selected list of repositories and numerical tools to which I have either directly built or made significant contributions to.

  • ESSBP: My python-based package capable of solving hyperbolic conservation laws with SBP and entropy-stable schemes in up to 3 dimensions. It is not particularly fast, but thanks to the wonders of JIT, not that slow either - Perfect for sandbox development!
  • I am a contributor and former code-manager of the in-house group code Jetstream, a Fortran code developed in the Computational Aerodynamics Lab at the University of Toronto, to solve the RANS equations using a Parallel Implicit Newton-Krylov Shur solver. For details regarding the code, we refer to either the group website above, or this publication.
  • Coming soon: a Julia repository for generating SBP operators on general function spaces.
